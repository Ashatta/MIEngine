CertTool is a small tool for adding vcremote's client cert to the windows certificate cache.

To use:
1. Set the hostname and port of vcremote in the source code.
2. Run 'vcremote generateClientCert' on the mac
3. Run 'vcremote' on the mac
4. Run CertTool
5. Enter 6 digit pin that vcremote displayed.
6. The certificate will be added to the windows cache.
